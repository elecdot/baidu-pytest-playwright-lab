# LaTeX Experiment Report

This directory contains the full LaTeX experiment report for the Baidu Map
Playwright automation lab.

## Build

Build the PDF from this directory:

```bash
make pdf
```

The build uses `latexmk` with `xelatex` because the report is written in
Chinese and uses `ctexart`.

## Output

The generated PDF is:

```text
reports/latex/main.pdf
```

The PDF and LaTeX intermediate files are ignored by git. The source files in
this directory are tracked.
